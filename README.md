### Football-Attendance-Predictions
# Emilio Barragán Godoy

The main goal of this proyect was to create a Machine Learning model that predicts the attendance of football matches all around the world. This included matches from every national league (like Liga MX or Premier League), every continental cup (like the UEFA Champions League or CONMEBOL Libertadores), as well as international tournaments like the World Cup, EURO, etc.
